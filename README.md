

# 荔枝UI
> 基于Vite4+TypeScript的Vue3组件库开发框架。**荔枝UI**是一个 **Vue3** 组件库开发环境框架，采用最新的 `Vite4+TypeScript` 为技术栈，支持`按需加载`、`单元测试`、`自动打包与发布`等功能。

## 目录结构
```angular2html
|- packages   组件库
|--- components
|- play       调试项目
|- site       静态文档站点

```

## 快速启动

- 安装 pnpm

```
npm install pnpm -g
```

- 安装依赖

```
pnpm install
```

- 打包组件库与工具库

```
pnpm run build:pro-ui
```

- 启动测试项目

```
pnpm run pro-ui:dev
```

- 启动静态文档站点

```
pnpm run docs:dev
```
